Sobre

Um pouco sobre como o projeto começou.


  • I

    2022

    O inicio

    Para fazer o estágio obrigatório do curso técnico precisavamos criar um projeto sobre programação, então veio a ideia de fazer um app que auxilie no aprendizado de programação, assim surgiu o Pyquiz.


  • II

    Criando o projeto

    Usando o Freamework React Native, começamos a montar o projeto, suas telas, programação, design, tudo isso sendo feito colaborativamente através do Github.


  • III

    Banco de dados

    O Pyquiz possui uma implementação do Firebase, em que é possivel logar no aplicativo e ter os dados do quiz salvo no banco de dados, tudo em tempo real.


  • IV

    Finalização

    Com todas as interfaces desenvolvidas e bugs sendo resolvidos, buscamos fazer a primeira build testável do projeto, podendo ser instalada nos celulares.

  • 2023
    Projeto
    Finalizado
